# AnrMonitor  

This is a tool for checking Android ANR problem.

本项目代码参考一下仓库，并对代码做了重构
[月光宝盒](https://github.com/xiaolutang/MoonlightTreasureBox)

第一版只支持将ANR日志保存在data/package/目录
第二版开发UI功能
